53 // If set, the underlying solver will call this function "regularly" in a
54 // deterministic way. It will then wait until this function returns with the
55 // current "best" information about the current problem.
56 //
57 // This is meant to be used in a multi-threaded environment with many parallel
58 // solving process. If the returned current "best" response only uses
59 // informations derived at a lower deterministic time (possibly with offset)
60 // than the deterministic time of the current thread, then the whole process can
61 // be made deterministic.

63  Model* model);
