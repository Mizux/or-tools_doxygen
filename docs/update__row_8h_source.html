26 // During a simplex iteration, when the basis 'leaving_row' has been
27 // selected, one of the main quantities needed in the primal or dual simplex
28 // algorithm is called the update row.
29 //
30 // By definition, update_row[col] is the coefficient at position
31 // 'leaving_row' in the current basis of the column 'col' of the matrix A.
32 //
33 // One efficient way to compute it is to compute the left inverse by B of the
34 // unit vector with a one at the given leaving_row, and then to take the
35 // scalar product of this left inverse with all the columns of A:
36 // update_row[col] = (unit_{leaving_row} . B^{-1}) . A_col
37 class UpdateRow {
38  public:
39  // Takes references to the linear program data we need.
40  UpdateRow(const CompactSparseMatrix& matrix,
41  const CompactSparseMatrix& transposed_matrix,
42  const VariablesInfo& variables_info, const RowToColMapping& basis,
43  const BasisFactorization& basis_factorization);

45  // Invalidates the current update row and unit_row_left_inverse so the next
46  // call to ComputeUpdateRow() will recompute everything and not just return
47  // right away.

50  // Computes the relevant coefficients (See GetIsRelevantBitRow() in
51  // VariablesInfo) of the update row. The result is only computed once until
52  // the next Invalidate() call and calling this function more than once will
53  // have no effect until then.
54  void ComputeUpdateRow(RowIndex leaving_row);
