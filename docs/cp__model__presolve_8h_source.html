187 // Presolves the initial content of presolved_model.
188 //
189 // This also creates a mapping model that encode the correspondence between the
190 // two problems. This works as follow:
191 // - The first variables of mapping_model are in one to one correspondence with
192 // the variables of the initial model.
193 // - The presolved_model variables are in one to one correspondence with the
194 // variable at the indices given by postsolve_mapping in the mapping model.
195 // - Fixing one of the two sets of variables and solving the model will assign
196 // the other set to a feasible solution of the other problem. Moreover, the
197 // objective value of these solutions will be the same. Note that solving such
198 // problems will take little time in practice because the propagation will
199 // basically do all the work.
200 //
201 // Note(user): an optimization model can be transformed into a decision problem,
202 // if for instance the objective is fixed, or independent from the rest of the
203 // problem.
204 //
205 // TODO(user): Identify disconnected components and returns a vector of
206 // presolved model? If we go this route, it may be nicer to store the indices
207 // inside the model. We can add a IntegerVariableProto::initial_index;
208 //
209 // Returns false if a non-recoverable error was encountered.
210 //
211 // TODO(user): Make sure this can never run into this case provided that the
212 // initial model is valid!
213 bool PresolveCpModel(const PresolveOptions& options,
214  CpModelProto* presolved_model, CpModelProto* mapping_model,
215  std::vector<int>* postsolve_mapping);
216 
217 // Replaces all the instance of a variable i (and the literals referring to it)
218 // by mapping[i]. The definition of variables i is also moved to its new index.
219 // Variables with a negative mapping value are ignored and it is an error if
220 // such variable is referenced anywhere (this is CHECKed).
221 //
222 // The image of the mapping should be dense in [0, new_num_variables), this is
223 // also CHECKed.
224 void ApplyVariableMapping(const std::vector<int>& mapping, CpModelProto* proto);
